GearCheck (Source Code)

Gearcheck is a World of Warcraft add-on that pulls data from Warcraftlogs and displays information about the best parses in a user-friendly way. The data should be used on a per-boss basis and should provide useful information as to how to perform as best as possible on specific encounters. For most users this should help on deciding the most useful trinkets and talents but it could be useful to also investigate the best stat combinations or rotations. Data is parsed through a python script and organized in encounter folders, then organized in class subfolders to make user-access easier.

State of the Project

The project is on hiatus right now waiting for Battle for Azeroth. The reason behind this is that the WoW API will probably change and make most addons broken so it would be counter productive to develop it with the current API and methods to then have to completely overhaul the code.

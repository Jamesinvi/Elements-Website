About the Project

Elements is a 3D adventure-platformer focused on a large scale world and fully fleshed out universe. There will be four main areas (based on the Elements) with a number of level/zones to complete and explore. All of the creatures and zones in this world are a rapresentation or combination of the four Elements: Fire, Water, Air and Earth.

Interactibles

My code is setup using mostly Inheritance with some abstract classes that define generic fields and methods for any given type of object. I am aware that some (if not most) games use Interfaces way more but I ultimately decided to use inheritance because I was more used to that style. So for interactible objects, I created an abstract class from which every object would derive from.
